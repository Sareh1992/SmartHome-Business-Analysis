# Smart Home Monitoring System – Business Analysis Project

## Project Overview
This project presents a complete business analysis documentation for a Smart Home Monitoring System.

The system is designed to monitor smart home devices, detect security-relevant events, generate alerts, and store activity logs for later analysis.

## Role
Business Analyst / Application Analyst

## Project Objectives
- Analyze business and system requirements
- Define functional and non-functional requirements
- Create user stories and use case descriptions
- Model the alert management process
- Prepare testing documentation
- Identify security and forensic risks
